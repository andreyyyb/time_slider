# Исторические периоды

Тестовое задание: Реализация блока с историческими периодами.
GitHub Pages: https://andreyyyb.github.io/time_slider/

## Технологии

- React + TypeScript
- SCSS
- Webpack
- GSAP (анимации)
- Swiper (слайдер)

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите проект в режиме разработки:
   ```bash
   npm start
   ```
   Проект будет доступен по адресу: http://localhost:9000

3. Сборка проекта для продакшена:
   ```bash
   npm run build
   ```
   Файлы сборки появятся в папке `dist`.

## Особенности реализации

- Блок полностью независим и может быть использован повторно.
- Количество точек на круге рассчитывается динамически (от 2 до 6).
- Анимация вращения круга реализована с помощью GSAP.
- Цифры годов анимируются при смене периода.
- Адаптивная верстка (мобильная версия упрощена для удобства использования).
