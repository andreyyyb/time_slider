# Исторические периоды

Тестовое задание: Реализация блока с историческими периодами.

## Технологии

- React + TypeScript
- SCSS
- Webpack
- GSAP (анимации)
- Swiper (слайдер)

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите проект в режиме разработки:
   ```bash
   npm start
   ```
   Проект будет доступен по адресу: http://localhost:9000

3. Сборка проекта для продакшена:
   ```bash
   npm run build
   ```
   Файлы сборки появятся в папке `dist`.

## Особенности реализации

- Блок полностью независим и может быть использован повторно.
- Количество точек на круге рассчитывается динамически (от 2 до 6).
- Анимация вращения круга реализована с помощью GSAP.
- Цифры годов анимируются при смене периода.
- Адаптивная верстка (мобильная версия упрощена для удобства использования).

Так как у меня нет прямого доступа к вашему GitHub, выполните следующие шаги:

    *   Зайдите на https://github.com/new

2.  **Инициализируйте Git в папке проекта**
    Откройте терминал в папке с проектом и выполните команды по очереди:

    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    ```

3.  **Свяжите локальный проект с GitHub**
    Скопируйте ссылку на ваш репозиторий с GitHub (она выглядит как `https://github.com/ваш-логин/historical-dates.git`) и выполните:

    ```bash
    git branch -M main
    git remote add origin https://github.com/ВАШ_ЛОГИН/ИМЯ_РЕПОЗИТОРИЯ.git
    git push -u origin main
    ```

    *Замените ссылку на ту, что скопировали.*
